# ペルソナ定義

更新日: 2026-02-07

## ターゲットユーザー
- 区分: B2C（個人利用のエンジニア中心）
- 属性: Vibe Coding を含む AI コーディングを日常的に行うエンジニア
- 利用シーン: エディタやチャットツール上で、キーボード入力の代わりに音声入力を高速に使いたい
- 競合サービス: Superwhisper

## ペルソナ（暫定）
- 名前: 日本人の AI Agent といっしょに開発しているエンジニア（仮）
- 立場: 個人開発者 / ソロエンジニア
- 課題:
  - 長文入力やプロンプト作成をキーボードだけで行うと時間がかかる
  - 既存の音声入力では開発向け入力（記号・英単語混在）が安定しない
- ゴール:
  - ショートカットで即時に録音開始し、正確な文字起こし結果を任意アプリに貼り付けたい
  - Mac/Windows/Linux のどこでも同じ体験で使いたい

## 未確定項目
