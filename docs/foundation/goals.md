# プロジェクトゴール

更新日: 2026-02-07

## 目的
- 現行プロジェクトを Go ベースへ完全移植する
- 最重要ゴールとしてマルチプラットフォーム対応を実現する

## 対象プラットフォーム
- Must: macOS / Windows / Linux
- Nice to have: iOS / Android
- モバイル方針: Desktop MVP 完了後に iOS / Android を検証（段階導入）

## MVP 定義（ユーザー合意済み）
- ユーザーがショートカットで話しかけると、正しく文字起こし結果が返る
- コピー・貼り付けが正しく実行される
- 現行プロジェクトで実現済みの体験を機能退行なしで移植する

## 要件メモ
- 音声入力システムは「既存入力へ重ねる」または「独自入力として切り替える」方式を想定
- メニューバー表現はプラットフォーム差分が大きく、移植時の主要検討ポイント
- 多言語展開は不要（日本語特化）
- 主要競合として Superwhisper を想定

## KPI（指標定義）
- 文字起こし成功率: **98%**
- 貼り付け成功率: **99%**
- 応答時間: **平均6秒以内**

## 未確定事項（Phase 0 以降で決定）
- 制約条件の具体化（現状は「技術的制約あり」のみ確定）
- iOS / Android での入力オーバーレイ実現方式

## 確定事項
- 事業区分: **B2C**
- UI実装方針: **Go中心（Desktop MVP先行）**
- GoのOS連携ライブラリ最終決定: **PoCで確定**
